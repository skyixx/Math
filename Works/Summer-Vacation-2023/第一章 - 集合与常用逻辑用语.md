# 题型一：元素与集合的关系

【例 1】已知集合$A = \{12,a^2+4a,a-2\}$, $-3 \in A$,则$a=(\ \ )$
答案：$D$

- A.$-1$
- B.$-3$或$-1$
- C.$3$
- D.$-3$

解：

- 1.$-3 = a^2+4a$

$$
\begin{align}
&\to(a+1)(a+3)=0\\
&\to a=-1 \ or \  a=-3\\
&if\ \ a = -1\\
&\to a-2 = -3\\
&\because \ \ a^2+4a=-3\\
&\therefore \ \ a\neq-1\\
&if \ \ a = -3\\
&\to a-2 = -5\\
&\therefore \ \ a = -3
\end{align}
$$

- 2.$-3 = a-2$

$$
\begin{align}
&a=-1\\
&\to a^2+4a = -3\\
&a\neq -1
\end{align}
$$

- 综上所述$a=-3$
